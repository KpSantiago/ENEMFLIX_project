Informações

O EVFLIX foi desenvolvido por Kauã Santiago, Guilhardy Matos, Maria Aparecida Fernandes e Leticia Liberato. O intuito do EVFLIX é servir como uma plataforma de estudos para os estudantes que estão se preparando para fazer a prova do enem, separando os melhores videos de estudo para cada diciplina que o Enem cobra.
